Example 1.1.7.

The value of a car dropped from $7400 to $6800 over the last year. What percent decrease is this?

To compute the percent change, we first need to find the dollar value change:

\begin{equation*} \$6800-\$7400 = -\$600. \end{equation*}

Often we will take the absolute value of this amount, which is called the absolute change:

\begin{equation*} \bigl| -600\bigr| = 600. \end{equation*}

Since we are computing the decrease relative to the starting value, we compute this percent out of $7400:

\begin{equation*} \frac{600}{7400}=0.081=8.1% \quad \text{decrease}. \end{equation*}

This is called a relative change.
